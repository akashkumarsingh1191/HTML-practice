List in HTML

'ul' tag is used for unordered list.

'li' tag is used for list item, it is placed inside the unordered list.

we can change the type of unordered list i.e. disc,cicle,square,none,etc. by using "type" attribute.

        Syntax --:
        ul tag type="bullet name"
        close ul tag.

Ordered lists in HTML :

"Ol"tag is used for Ordered List

"li"tag is used for list item inside a Ordered list.

We can change the bullets of ordered list i.e. (1,I,A,a,i) by using "type" attribute
